## Como usar este arquivo em uma sessão nova do Claude Code

1. Abrir o Claude Code no diretório do repo
2. Copiar a seção da trilha desejada (incluindo os 20 tópicos)
3. Colar num prompt iniciado com:

> "Continue o projeto Czarnikow. Leia o arquivo `czarnikow-tracks-backlog.md`
> para contexto e a última lição em `courses/czarnikow/course.json` para
> ver o formato exato. Gere a trilha `[ID DA TRILHA]` completa — 20 tópicos
> × 3 níveis (Confidence, Rise, Apex) = 60 lições. Use o ID de trilha
> correto (ex: `uk-england`, `supply-chain`, `logistics`, `accounting`,
> `fiscal-taxes`, `it`). Insira as lições com Edit em batches de 5,
> mantendo o JSON válido. Ao final, valide com Node que o total de
> lições bateu."
